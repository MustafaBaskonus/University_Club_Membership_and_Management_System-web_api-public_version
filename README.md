# Üniversite Kulüpleri Üyelik ve Yönetim Sistemi ( Web API Tabanlı Kurumsal Uygulama)
Çok rollü (Multi-Role) yetkilendirme ve yetki hiyerarşisine sahip; kulüplerin üyelik ve faaliyetlerini yönetmek için geliştirilmiş bir Web API projesidir.

Roller: Admin, Akademisyen, Kulüp Yöneticisi (Club Manager) ve Kullanıcı (User) olmak üzere dört temel rol mevcuttur.
Rol Yetkileri:
1. Admin: Tüm sistemin ve tüm kulüplerin genel yönetimi.
2. Akademisyen: Danışmanı olduğu kulüplerin üst yöneticisi; tüm işlemleri yönetme ve onaylama/reddetme yetkisi.
3. Kulüp Yöneticisi: Kendi kulübünü yönetir; duyuru ve etkinlik gibi kritik işlemler için Akademisyen veya Admin onayı gereklidir.
4. Kullanıcı: Profilini yönetir ve kulüplere üyelik işlemlerini gerçekleştirir.
